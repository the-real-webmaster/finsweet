# 🌐 Finsweet — Multi‑Page Website

Многостраничный адаптивный сайт, созданный с использованием HTML5, CSS3 и JavaScript. Проект демонстрирует навыки верстки, организацию нескольких страниц, работу с блогом, контактной формой и авторской страницей.

## 🔗 Ссылка
[Посмотреть проект](https://the-real-webmaster.github.io/finsweet/)

## 🛠 Технологии
- HTML5  
- CSS3  
- JavaScript  
- Адаптивный дизайн под разные устройства  

## 📂 Страницы сайта
- `index.html` — главная страница  
- `about-us.html` — страница «О нас»  
- `author.html` — страница автора  
- `blog.html` — блог  
- `blog-post.html` — отдельный пост блога  
- `category.html` — категории блога  
- `contact.html` — контакты  
- `privacy-policy.html` — политика конфиденциальности  

## 📌 Особенности
- Многостраничная структура с корректными ссылками между страницами  
- Адаптивный дизайн для мобильных, планшетов и десктопов  
- Семантическая верстка и организованные CSS‑стили  
- Контактная форма и разделы блога с отдельными страницами  
- Простая и понятная навигация по сайту
